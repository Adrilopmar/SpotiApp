Features

Dark Mode

It is always a good idea to implement diferents palettes to your web in order to let the user decide which mode to use.

Light Mode

There are a lot of reasons to introduce dark mode. The mains reasons to set up this feature were to learn how dark/light mode work and reduce eye strain.

Async () => {{'{ Await }'}}

This Java Script feature is used to make functions asynchronous. It is not rare to need to wait a result before executing the next function. Sounds logic right? So if you do not use async and await function sometimes you get errors.

One of the purpose of this project was learning how to Async / Await work. I may say that after several errors and hours of hard work, I am familiarized with this Java Script feature.

Linked to Spotify API

Fetching data and working with API (Application Programming Interface) is really common in front-end world. So I built this web Application to learn and get used to work with APIs.

Spotify API have been a great practice where I have worked with headers and authorization's tokens for the purpose of fetching API's data.
